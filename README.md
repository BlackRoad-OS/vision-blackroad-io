## 📦 Overview

**vision-blackroad-io** is the Computer Vision module of the BlackRoad OS platform, providing image and video analysis capabilities for authorized BlackRoad ecosystem integrations.

### ✨ Features

- 🎯 Image and video analysis pipeline
- 🔐 OAuth 2.0 secured endpoints (see [OAUTH.md](OAUTH.md))
- 🌐 Cloudflare-powered edge delivery
- 💳 Stripe-integrated billing
- 🛡️ Converter API access gate for contributors

---

## 🚀 Getting Started

### Prerequisites

- BlackRoad Converter API key (required — see [CONTRIBUTING.md](CONTRIBUTING.md))
- Node.js 18+
- Cloudflare Wrangler CLI

### Installation

```bash
npm install
npm run dev
```

### Deployment

```bash
npm run deploy
```
